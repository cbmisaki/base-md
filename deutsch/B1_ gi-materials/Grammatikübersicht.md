
### 01. zusammengesetzt Adjektive
---------------------------------------------------------------------------

- **adj + adj**
-> hell + blau = *hellblau*
-> süß + sauer =  *süßsauer*

- **N + adj**
-> Stunden (Pl) + lang = *stundenlang*
-> der Alkohol + frei = *alkoholfrei*
